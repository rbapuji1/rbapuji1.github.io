A few of my best coding projects

Edmonton Reads

Built using Django, two of my friends and I created this website. It sources the top 10 most read books in Edmonton every week and displays them on the website. We also used a web scraper to gather descriptions and images for each book. We also added functionality for user accounts and personalized reading lists.

Visit site GitHub repository

RateMyTracks

Using JavaScript, HTML, and Python, we created a Google Chrome Extension to help with choosing courses for university. This extension gathers ratings for each professor from RateMyProfessors and displays them within a course choosing page. Currently the extension has more than 150 installs.

Chrome Webstore GitHub repository

Other Projects

When I started out with programming, I made a few short scripts to solve puzzles and problems I was interested in. Using Python, I solved sudoku and the Towers of Hanoi puzzle using recursion.
